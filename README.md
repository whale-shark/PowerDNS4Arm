# PowerDNS for Armhf

Minimal PowerDNS and PowerDNS-Admin for ARM.

Backend databease for sqlite3.

Application image

```
Client computer
  \/
PowerDNS Recursor -> local domain -> PowerDNS Authoritative Server 
  \/
other than local domain
  \/
Internet
```


### PowerDNS

```

```

### PowerDNS-Admin

```
power_dns_admin/sqlite/pdns_admin_table.db
```
